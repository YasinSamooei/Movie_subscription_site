### :gear: Installation

setup project
```bash
docker-compose up
```
build project
```bash
docker-compose up --build -d
```


### :test_tube: Running Tests

pytest
```bash
docker-compose exec backend sh -c "pytest ."
```
use black
```bash
docker-compose exec backend sh -c "black ."
```
